# abc_line_renders

```
usage: render_step_file.py [-h] -i INPUT_FILE -o OUTPUT_FILE [--width WIDTH]
                           [--height HEIGHT] [--line-width LINE_WIDTH]
                           [--margin-top MARGIN_TOP]
                           [--margin-left MARGIN_LEFT] [--loc LOCATION]
                           [--dir DIRECTION] [--export-hidden-edges]

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT_FILE, --input-file INPUT_FILE
                        input dir with ABC dataset.
  -o OUTPUT_FILE, --output-file OUTPUT_FILE
                        output file.
  --width WIDTH         render width.
  --height HEIGHT       render height.
  --line-width LINE_WIDTH
  --margin-top MARGIN_TOP
                        render margin on top.
  --margin-left MARGIN_LEFT
                        render margin on left.
  --loc LOCATION        projection hyperplane normal origin
  --dir DIRECTION       projection hyperplane normal direction
  --export-hidden-edges
                        render hidden edges as dashed lines
```
